title=[B41/B42] Elyon Lib
description=[h1]Elyon Lib[/h1]
description=
description=Shared [b]Build 41[/b] utility library: logging, timers, JSON/file I/O, table helpers, networking glue, rewards, dates, colours, sprite property helpers, layout/theme building blocks, the [b]Dock Menu[/b] (edge-pinned launcher other mods register into), and [b]HUD notifications[/b] (toast stack beside vanilla moodles for mods that opt in).
description=
description=[h1]Who needs it[/h1]
description=[list]
description=[*][b]Players:[/b] install Elyon Lib only when a mod you use lists it as a dependency. Load it [b]above[/b] those mods in the mod list.
description=[*][b]Modders:[/b] `require("ElyonLib/...")` the pieces you need from shared or client paths (see GitHub README / GUIDE).
description=[/list]
description=
description=[h1]What you get (high level)[/h1]
description=[list]
description=[*][b]Core:[/b] per-mod loggers with levels, table pretty-print, optional client/server log relay, and a central timer queue (MP server uses one-minute ticks; client uses per-frame updates).
description=[*][b]Data & files:[/b] buffered read/write, JSON parse/stringify, deep/shallow table copy, merge, and safe debug formatting with circular-reference detection.
description=[*][b]Game integration:[/b] player keys and online player lists, admin access-level ordering, item display names/textures with caching, perk resolution, and network-safe item/XP grants for reward-style mods.
description=[*][b]UI:[/b] text measure/trim/wrap, frame-step smoothing, IS element helpers, window layout (centre, clamp, default geometry), shared dark theme palette, scrollable panel pieces, calendar selector, [b]MenuDock[/b] (the Dock Menu rail players pin to the screen edge), and [b]HudNotify[/b] (right-side toast stack for optional notifications).
description=[*][b]World/building:[/b] sprite property set/unset helpers and batch updates on the sprite manager.
description=[*][b]Time:[/b] timezone offsets, date keys, timestamps, parsing/formatting (ISO/US/EU), comparisons, and PZ calendar hooks.
description=[/list]
description=
