title=[B41] Elyon Lib
description=This is a utility framework that provides essential tools and functions. It's designed to streamline common development tasks and provide a solid foundation for other mods.
description=
description=[b]Core Features:[/b]
description=[list]
description=[*][b]Advanced Logging System[/b] - Configurable logging with multiple levels (ERROR, WARNING, INFO, DEBUG) and mod-specific loggers
description=[*][b]Timer Management[/b] - Robust timer system supporting milliseconds, seconds, and minutes with loop functionality
description=[*][b]File Utilities[/b] - Comprehensive file I/O operations including JSON parsing, buffered reading, and error handling
description=[*][b]Table Formatting[/b] - Pretty-print tables with circular reference detection for debugging
description=[*][b]Sprite Property Management[/b] - Tools for managing sprite properties and flags efficiently
description=[/list]
description=
description=[b]For Developers:[/b]
description=ElyonLib provides a clean, well-documented API that makes mod development faster and more reliable. It handles common edge cases and provides consistent error handling across all utilities.
description=
description=[b]For End Users:[/b]
description=This is a framework mod that other mods may require. You only need to install it if a mod you use depends on it. ElyonLib itself doesn't add gameplay content.
description=
description=[b]Technical Details:[/b]
description=[list]
description=[*]Built for Build 41 compatibility
description=[*]Optimized for both single-player and multiplayer
description=[*]Lightweight and efficient
description=[*]Follows Project Zomboid modding best practices
description=[/list]
